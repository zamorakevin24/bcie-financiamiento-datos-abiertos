This repository contains an analysis of the financing provided by the Central American Bank for Economic Integration (BCIE), using open data on loan approvals and disbursements between 1960 and 2025.

The project was developed as part of the Journalism and Research for Development Competition â€“ BCIE Open Data (2026).

## Scope of the analysis

The analysis covers:

- Historical evolution of approved and disbursed amounts by year.
- Comparison between approvals and disbursements by decade.
- Identification of the Top 5 countries with the highest approved and disbursed amounts.
- Distribution of financing by institutional sector (public and private).
- Sectoral evolution of financing over time.

## Tools used

- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Jupyter Notebook

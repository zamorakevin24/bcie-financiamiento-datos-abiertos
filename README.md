# Análisis del financiamiento del BCIE con datos abiertos (1960–2025)

Este repositorio contiene el análisis del financiamiento otorgado por el Banco Centroamericano de Integración Económica (BCIE), utilizando datos abiertos de aprobaciones y desembolsos entre 1960 y 2025.

El proyecto fue desarrollado como parte del **Concurso de Periodismo e Investigación para el Desarrollo – Datos Abiertos BCIE (2026)**.


## Alcance del análisis

El análisis aborda:

- Evolución histórica de los montos **aprobados** y **desembolsados** por año.
- Comparación entre aprobaciones y desembolsos por **década**.
- Identificación de los **Top 5 países** con mayores montos aprobados y desembolsados.
- Distribución del financiamiento por **sector institucional** (público y privado).
- Evolución sectorial del financiamiento a lo largo del tiempo.


## Herramientas utilizadas

- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook
